# Smart File Organizer Using Python

## Description

This project automatically organizes files into different folders based on their file extensions.

## Features

- Organizes Images
- Organizes Documents
- Organizes Videos
- Organizes Audio Files
- Organizes Archives
- Creates folders automatically

## Technologies Used

- Python
- os module
- shutil module

## How to Run

1. Download the project.
2. Run FileOrganizer.py
3. Enter folder path.
4. Files will be organized automatically.
